Operation Pause Tool
====================

Adds a toolbar cursor tool (unlocked with vanilla Tools research) that pauses building operation only. Construction is not affected.

Controls:
- Left click / drag: pause operation
- Right click / drag: unpause operation
- Ctrl: include transports (belts, pipes)

Install: build the mod or copy the release folder to:
%APPDATA%\Captain of Industry\Mods\OperationPauseTool\

Toolbar: appears next to the vanilla Pause tool after Tools research (PauseOnly icon).
No dedicated keyboard shortcut — click the toolbar button. Vanilla pause tool uses P (TogglePauseTool).

Requires Captain of Industry 0.8.3+ (verified on 0.8.4b).

Set COI_ROOT to your game install before building from source.
